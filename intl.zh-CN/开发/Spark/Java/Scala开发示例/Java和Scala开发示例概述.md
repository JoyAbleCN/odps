# Java和Scala开发示例概述

本文为您介绍使用Java和Scala语言开发的示例。

## 下载示例工程

MaxCompute Spark提供了项目示例工程模板，建议您下载复制后直接在模板里进行开发。

执行如下命令下载示例工程。

```
# spark-1.x 模板及编译  
git clone https://github.com/aliyun/MaxCompute-Spark.git  
cd spark-1.x  
mvn clean package  
# spark-2.x 模板及编译  
git clone https://github.com/aliyun/MaxCompute-Spark.git  
cd spark-2.x  
mvn clean package
```

**说明：** 示例工程里关于Spark依赖的Scope为provided，此参数请不要更改，否则提交的作业无法正常运行。
